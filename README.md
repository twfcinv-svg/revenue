
# 供應鏈月營收熱力圖（上下游 Treemap 版）

- **查詢結果卡片**顯示在查詢按鈕右側：代號｜中文名稱、產業、月份/指標與數值。
- **下方版面兩欄等寬**：左＝上游 Treemap、右＝下游 Treemap，皆依 Links 的「關係類型」分群；每檔個股以矩形**包起來**（熱力圖色塊），標示「代號、中文名稱、數值」。
- 若某群組只有一檔個股，視覺會較像單一大方塊，屬正常狀況（受測試資料關聯數量影響）。

## Excel 規格
- **Revenue**：`個股`、`名稱`、`產業別`，以及 `YYYYMM單月合併營收年成長(%)`、`YYYYMM單月合併營收月變動(%)`。
- **Links**：`上游代號`、`下游代號`、`關係類型`（本版本**不使用**「權重」）。

## 部署
1. 將 `index.html`, `styles.css`, `app.js`, `data.xlsx` 放在 repo 根目錄。
2. GitHub Pages：Settings → Pages → Deploy from a branch → 選 `main` 與根目錄 `/`。

## 顏色邏輯
- 可切換 **紅=好 / 綠=弱**（預設）或 **綠=好 / 紅=弱**。
- 面積用 |MoM/YoY| 表示動能；顏色透明度隨強度提升。
